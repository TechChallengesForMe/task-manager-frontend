# HMCTS Task Manager Frontend

## What is this?
Web interface for caseworkers to manage tasks.

## How to run:
1. Start the backend first (port 8080)
2. Open `index.html` in your browser
3. Or run: `python3 -m http.server 3000`

## Features:
- Create, view, edit, delete tasks
- Filter by status
- Task statistics

## Technologies:
- HTML, CSS, JavaScript, jQuery
